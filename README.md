# SSAR

This repository provides code and experimental results for the paper "SSAR: A Novel Software Architecture Recovery Approach Enhancing Accuracy and Scalability".

The *Baseline* folder stores the baseline implementation.

The *Raw\_Data* folder stores the data including ground\_truth files, dependency files, and the recovered architectures.

The *SSAR* folder stores SSAR's code files, along with detailed usage instructions and running examples.
